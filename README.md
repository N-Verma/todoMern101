# Todo App

The Todo app allows addition of task, due date and tag to these task.
Every Task can be chekced out or deleted.

## How to install it
1) `git clone` the repo
2) `npm install`
3) `nodemon start`
4) `cd client` After that  `npm install`
5) `npm run start`
